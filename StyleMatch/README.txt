StyleMatch iOS App

Team Members:
S4027498 (Manan Chaudhary)
GitHub Repository:
https://github.com/rmit-iPSE-s2-2025/StyleMatch

External Resources Used:
- iOS Simulator (iOS 16/17)
- Xcode 15+
- SwiftUI Framework
- SF Symbols (Apple's icon library)
- Local JSON data (products.json)

Run Instructions:
1. Open StyleMatch.xcodeproj in Xcode
2. Select iOS 16/17 simulator as target device
3. Press Cmd + R or click the Play button to build and run
4. The app will launch in the iOS Simulator

App Features:
- Home/Explore screen with product catalog
- Scan/Reference Presets for style matching
- Results screen showing matched items
- Item Detail view for product information
- Saved items functionality
- Custom components: Wrap, TagChip, ProductCard
- SwiftUI Navigation with NavigationStack and NavigationLink
- Local data source using products.json
